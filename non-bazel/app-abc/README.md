# App ABC

A simple JavaScript greeting application.

## Requirements

- Node.js: 24.4.1
- pnpm: 10.13.1

## Installation

```bash
pnpm install
```

## Usage

Run the application:

```bash
pnpm start
```

Or run with watch mode:

```bash
pnpm dev
```

## Features

- Simple greeting functionality
- System information display
- Timestamp-based messages
- Command-line argument parsing with minimist

## ⚠️ Security Note

This application includes **intentionally vulnerable dependencies** for security testing purposes. See `../SECURITY_VULNERABILITIES.md` for details.

To check vulnerabilities:
```bash
pnpm audit
```

**DO NOT use in production** without updating dependencies.
